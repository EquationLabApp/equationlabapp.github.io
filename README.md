## Custom Domain Setup

To connect a custom domain (e.g., equationlab.com):
1. Purchase domain from registrar
2. Add domain to Cloudflare (free)
3. Create CNAME record pointing to equationlabapp.github.io
4. Add custom domain in GitHub repo Settings > Pages
5. Enable Enforce HTTPS
